### Insert 

```sql
insert into movie (title, year, genre)
values ('haskell', 2021, 'aids')
```

or well you could reuse data from db using subqueires. 

```sql
insert into studio (name)
select distinct studio from movies m 
where m.studio not in (select s.name from studio s)
```

### Delete

```sql
delete from movie 
where length > 180 or (title, year) in (
	select title, year from casting 
	where name = "Keanu Reeves"
)
```

basically the above queries deletes all the movies that are longer than 180 mins or the movies in which Keanu Reaves acted in.
